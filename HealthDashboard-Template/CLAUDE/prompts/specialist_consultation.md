# Шаблон: Консультация специалиста

## Когда использовать

Когда нужен анализ с точки зрения конкретной специальности.

## Промпт

```
Прочитай файл CLAUDE/agents/[специальность].md и проведи анализ всех данных из 02_TABLES/.

Специальности: cardiologist / endocrinologist / gastroenterologist / pain_specialist

Сохрани отчёт в 03_AGENT_REPORTS/YYYY-MM-DD_[специальность].md
```
