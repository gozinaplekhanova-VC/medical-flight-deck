# Шаблон: Анализ новых результатов

## Когда использовать

Когда в `00_INBOX_RAW/` появились новые документы.

## Промпт

```
У меня появились новые медицинские документы в папке 00_INBOX_RAW/.

Пожалуйста:
1. Прочитай CLAUDE/agents/sorter.md и отсортируй документы
2. Прочитай CLAUDE/agents/extractor.md и извлеки показатели в CSV
3. Прочитай CLAUDE/agents/normalizer.md и нормализуй данные
4. Затем выбери релевантных агентов-специалистов из CLAUDE/agents/ и проведи анализ
5. Обнови дашборд: python3 scripts/build_timeline.py && python3 scripts/generate_dashboard.py

После каждого шага сообщай о результатах.
```
