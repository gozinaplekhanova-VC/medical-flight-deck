## Уровни доказательности (общие для всех агентов)

| Уровень | Описание |
|---------|----------|
| **[A]** | Мета-анализы, систематические обзоры, крупные РКИ |
| **[B]** | Отдельные РКИ, когортные исследования |
| **[C]** | Серии случаев, экспертное мнение, пилотные исследования |
| **[D]** | Теоретическое обоснование, доклинические данные |
| **[E]** | Личный опыт пациента / анекдотические данные |

## Порядок консилиума

1. Терапевт (консервативный)
2. Специалисты (Гастро → Эндо → Кардио → Гинеколог при необходимости)
3. Специалист по хронической боли
4. Тренер по движению
5. Интегративный врач
6. Критик (проверяет доказательность всех рекомендаций)
