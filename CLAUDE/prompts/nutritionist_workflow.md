# Флоу работы с нутрициологом

## Структура папок

```
health_dashboard/
├── agents/
│   └── agent_nutritionist.md          ← описание агента
├── data/
│   ├── food_profile.csv               ← блюда, предпочтения, исключения
│   ├── food_ingredients.csv           ← инвентарь ингредиентов
│   ├── food_diary.csv                 ← лог приёмов пищи (экспорт из логгера)
│   ├── symptoms_diary.csv             ← ЖКТ-симптомы
│   ├── cycle_tracker.csv              ← трекер цикла
│   ├── blood_biochemistry.csv         ← биохимия крови
│   ├── blood_general.csv              ← общий анализ крови
│   ├── vitamins.csv                   ← витамины
│   ├── hormones.csv                   ← гормоны
│   ├── genetics.csv                   ← генетика
│   ├── medications.csv                ← медикаменты
│   └── supplements.csv                ← добавки
├── requests/
│   └── YYYY-MM-DD_nutrition_request.md ← запросы к агенту
├── plans/
│   └── meal_plan_current.md           ← текущая ротация
├── logs/
│   └── nutrition_journal.md           ← журнал: что работало, что нет
└── tools/
    └── food_diary_logger.jsx          ← фото-логгер (артефакт)
```

## Цикл работы

### 1. Запрос → Ротация (раз в 2–3 недели)

Создаёшь файл в `requests/`:

```markdown
# Запрос нутрициологу — 2026-03-25

## Текущее состояние ЖКТ
- Рефлюкс: активный / ремиссия / частичная ремиссия
- Вздутие: частое / редкое / нет
- Общее: ...

## Цели на этот цикл
- Например: увеличить белок, снизить простые углеводы, попробовать X

## Что изменилось
- Новые анализы: ...
- Новые добавки/медикаменты: ...
- Реакции на предыдущую ротацию: ...

## Что хочу
- Новая 5-дневная ротация / корректировка текущей / замена 2 дней
```

Затем просишь Claude: *"Запусти нутрициолога на основе нового запроса в requests/"*

### 2. Нутрициолог генерирует ротацию

Результат сохраняется в `plans/meal_plan_current.md`:

```markdown
# Ротация — 25.03–08.04.2026

## Фаза: частичная ремиссия рефлюкса
## Цель: стабильная глюкоза, увеличение белка

### День 1: Средиземноморский
- Завтрак: ...
- Обед: ...
- Ужин: ...
- ЖКТ-ремарки: ...
- Гликемические заметки: ...

### День 2: Русская классика
...

### День 5: ...
...

## Безопасные замены при обострении
- Вместо X → Y
- Вместо A → B
```

### 3. Проживание ротации (~15 дней)

- Каждый из 5 дней ешь по 2–3 раза
- Логируешь еду через фото-логгер (фото → КБЖУ → оценки)
- Заполняешь symptoms_diary при необходимости

### 4. Фидбек скопом

После 15 дней экспортируешь CSV из логгера и пишешь сводку:

```markdown
# Фидбек на ротацию — 08.04.2026

## День 1
- Завтрак: отлично, повторяю
- Обед: тяжеловат, вздутие 2/3 оба раза
- Ужин: заменила X на Y — стало лучше

## День 3
- Не зашёл целиком — слишком много бобовых

## Общее
- Энергия стабильная
- Рефлюкс не обострялся
- Хочу больше рыбы, меньше пасты
```

### 5. Нутрициолог корректирует

На основе фидбека + данных из food_diary.csv:
- Обновляет `meal_plan_current.md`
- Пишет в `nutrition_journal.md` что работало / не работало
- Обновляет `food_profile.csv` если появились новые данные

Цикл повторяется.
