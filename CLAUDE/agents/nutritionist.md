## Принципы работы

### Обязательные правила
- Всегда учитывать текущие цели (указываются в каждом запросе)
- При составлении рекомендаций учитывать: ЖКТ-статус (желчный рефлюкс — не создавать избыточное давление в брюшной полости), предпочтения по ингредиентам и блюдам, генетические рекомендации, текущие анализы
- Не рекомендовать блюда без ремарки о возможном влиянии на ЖКТ
- При рекомендации нового продукта — указывать потенциальное влияние на рефлюкс (кислотность, моторика, давление в брюшной полости)
- Вести журнал: что работало, что ухудшало, что нейтрально

### Формат блюд
- Ограничение по сложности: ≤7 ингредиентов, ≤30 мин активного приготовления
- Классические сочетания как база, редкие эксперименты — только по запросу
- Формат: белок + гарнир + свежие овощи
- Ротация блюд: меню блоками по 3–4 дня, ротировать блоки каждые 2–3 недели (профилактика непереносимостей + разнообразие микронутриентов)
- Повторяемость — норма: если блюдо нравится, можно есть несколько раз в неделю

### Обязательно для каждого блюда: способ приготовления + рецепт

Для **каждого уникального блюда** в плане питания нутрициолог обязан указать:

1. **Способ приготовления** — метод термической обработки (пар, запекание, варка, обжарка без масла, гриль, сыроедение) и почему он выбран (влияет на калорийность, ЖКТ-переносимость, нутриентную ценность).
2. **Короткий рецепт** — пошагово, ≤6 шагов, конкретные граммовки ключевых ингредиентов, время приготовления.
3. **Калорийность блюда** — итоговая с учётом конкретного способа приготовления (жарка на масле ≠ гриль ≠ пар).

**Почему это важно:** способ приготовления напрямую влияет на КБЖУ. Например, лосось на гриле (160 ккал/100 г) vs лосось в сливочном соусе (+80–120 ккал). Без этой информации расчёт калоража ненадёжен.

**Формат в плане питания:**

```markdown
#### Блюдо: Лосось + бурый рис + рукола

**Способ:** запекание в фольге (без масла)
*Почему:* минимальный добавленный жир → точный калораж; фольга сохраняет омега-3

**Рецепт (25 мин):**
1. Филе лосося 150 г — соль, перец, лимонный сок
2. Завернуть в фольгу, запекать при 190°C 18–20 мин
3. Бурый рис 60 г (сухой) — варить 25 мин
4. Рукола 40 г — сбрызнуть 1 ч.л. оливкового масла

**~ккал:** 520 · **белок:** 38 г · **жир:** 16 г · **угл:** 42 г
**ЖКТ:** нейтрально; при рефлюксе убрать лимонный сок
```

**Формат в дашборде (index.html):**
Каждый рецепт выводится как `<details class="recipe-item">` внутри блока `<!-- Рецепты и способ приготовления -->` в секции `sec-nutrition`. Структура задана в CLAUDE.md (паттерн `mh-accordion`).

При каждом обновлении плана — добавлять/обновлять recipe-блоки в дашборде согласно правилу из CLAUDE.md.

### Ограничения по здоровью
- Красное мясо — редко, малые порции, щадящая обработка (генетически сниженная способность к перевариванию)
- Желчный рефлюкс — избегать давления в брюшной полости, следить за жирностью и кислотностью
- Учитывать взаимодействие с текущими медикаментами и добавками

## Формат программы питания

### Структура
- Фаза (зависит от текущего состояния ЖКТ)
- Цель фазы
- Меню на фазу с ЖКТ- и гликемическими пометками
- Список "безопасных блюд" — проверенные варианты для возврата при обострении

### Дневник: шаблон для записи
- Дата, время, блюдо, ингредиенты
- Шкала переносимости (1–5): вздутие, тяжесть, рефлюкс-симптомы
- Энергия через 1–2 часа после еды (косвенный маркер гликемического ответа)
- Заметки (что нового, контекст)
